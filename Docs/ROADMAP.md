# Marq'd — Public Roadmap

> *This is where we are. This is where we're going. This is what it will take.*

Marq'd is in active development. This roadmap is honest about what exists, what's being built, and what's aspirational. It is a public document — contributors, partners, and supporters should be able to see exactly where the project stands.

Last updated: 2025

---

## Current Status — Founding Site ✅

The Marq'd founding site is live. It establishes the project, communicates the mission, and invites early supporters and contributors.

**What exists now:**
- Interactive 3D globe with representative seed data
- Full mission and about pages
- Contact and support pages
- Open Collective donation infrastructure
- GitHub repository with full architectural documentation

**What this stage is for:**
Building awareness, finding collaborators, and establishing Marq'd as a serious and credible project before the full archive goes live.

---

## Phase 1 — The Archive ⚙️ *In Development*

**Goal:** A fully functional, live archive. Real stories. Real coordinates. Permanent records.

### Features
- [ ] Live story submission — anonymous, no account required
- [ ] Supabase backend — stories stored, retrieved, displayed on the globe in real time
- [ ] Archive entry format — record numbers, permanent IDs, full card structure
- [ ] AI-read audio — ElevenLabs integration, 4 voice options
- [ ] Photo submission — places and spaces only, moderation queue
- [ ] Reactions — no account required
- [ ] Age gate — confirmed on first visit, stored locally
- [ ] Rate limiting — one submission per IP per hour
- [ ] "Cite this entry" — one-click academic citation copy
- [ ] Mobile-optimized submission flow
- [ ] Content moderation — report button, review queue

### What Phase 1 Looks Like When Done
A stranger anywhere in the world can open Marq'd, find a location on the globe, read real stories from that place, listen to them if audio was generated, and submit their own story — anonymously, safely, permanently — in under five minutes.

## Phase 2 — The Community 🔭 *Planned*

**Goal:** Optional identity, social connection, and deeper engagement — without compromising the anonymous core.

### Features
- [ ] Optional user profiles — display name, pronouns, city, bio, avatar
- [ ] Per-story privacy control — anonymous or attributed, decided at submission
- [ ] Follow other users (public profiles only)
- [ ] Near Me feed — stories within user-defined radius
- [ ] User voice uploads — contributor records their own reading (with safety guidance)
- [ ] Threaded comments on archive entries
- [ ] Direct messaging — mutual follows only, no cold contact
- [ ] "Get Involved" contributor profiles — for people who want to be publicly associated with the project
- [ ] Moderation dashboard — admin tools, flagged content review
- [ ] Ad slot infrastructure — queer-friendly brand partners, region-aware

---

## Phase 3 — The Institution 🔭 *Aspirational*

**Goal:** Marq'd becomes a recognized resource for researchers, historians, journalists, and human rights advocates.

### Features
- [ ] Time-scrubber — watch the archive grow chronologically on the globe
- [ ] Legal context layer — overlay showing LGBTQ+ legal status by country (powered by ILGA World / Equaldex data)
- [ ] Research export — open-format archive download (JSON, CSV) for scholars
- [ ] Periodic archival snapshots — mirrored to the Internet Archive
- [ ] Translations — submission flow and UI in at least 10 languages
- [ ] Tor-friendly access — documented safe access path for high-risk contributors
- [ ] Institutional partnerships — ONE Archives, ILGA World, OutRight Action International
- [ ] Academic citation standard — formal integration with Zotero, Mendeley
- [ ] Grant funding — applications to Arcus Foundation, Gill Foundation, NEH
- [ ] Light mode — for accessibility and reading preference
- [ ] API access — for researchers and partner organizations

## How to Help Right Now

Marq'd is a solo founder project in its earliest stage. The areas where outside help matters most:

**Development**
React, TypeScript, Supabase, globe.gl, ElevenLabs API integration. If you build things and believe in this, reach out.

**Design**
The visual system is documented in `docs/DESIGN.md`. A second set of eyes from someone with strong UI/UX instincts would be valuable.

**Archival & Academic**
If you work in LGBTQ+ archives, digital humanities, oral history, or human rights documentation — the work you do informs everything about how this archive should be structured. We'd love your perspective.

**Community & Outreach**
Knowing the right people in the right organizations is how Marq'd becomes something that lasts. If you have relationships with LGBTQ+ organizations, academic institutions, or queer media — let's talk.

**Legal**
Privacy policy, terms of service, operating in hostile jurisdictions, nonprofit structure. If you have legal expertise and want to contribute it, it would be meaningful.

**Languages**
The archive needs to be accessible in Arabic, Farsi, Russian, Mandarin, Portuguese, Spanish, French, Hindi, Swahili, and more. If you can contribute translation work, you're helping reach the people who need this most.

## A Note on Timeline

There is no artificial urgency here. Marq'd will be built carefully, correctly, and with the people it serves in mind. A rushed archive is a flawed one. A flawed archive lets people down.

That said — every day without a live submission form is a day someone's story goes unrecorded. We are moving with purpose.
